# AI-Powered Meeting Summarizer

A command-line tool that converts audio recordings of meetings into transcripts and provides concise summaries using `whisper.cpp` for audio-to-text conversion and `Ollama` for text summarization.

## Features
- Audio preprocessing with FFmpeg
- Transcription using Whisper.cpp (`small` model)
- Summarization via Ollama server API

## Requirements
- Python 3.x
- FFmpeg
- whisper.cpp
- Ollama server running locally
